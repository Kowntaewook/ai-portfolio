D+ (D-Plus)

불법 웹사이트 및 특정 웹사이트 사용자 색출 시스템



📌 프로젝트 개요

불법 웹사이트를 아무런 거리낌 없이 이용하는 사용자들이 늘어나고, 적발된 운영자에게 오히려 동정 여론이 생기는 현상에 문제의식을 가져 개발을 시작했습니다.

학생들의 불법 웹사이트 이용을 억제하고, 교내 게시글 플랫폼(디시인사이드 등)에서 악성 댓글 작성자를 색출하기 위한 백신형 모니터링 프로그램입니다.

🛠️ 주요 기능

1\. 브라우저 기록 모니터링

특정 블랙리스트 URL(예: 디시인사이드)에 마지막으로 접속한 시간을 기준으로, 이후 방문한 URL 전체를 서버에 보고

보고 시각도 함께 기록되어 추적 가능

2\. 암호화 저장

수집된 기록은 \*\*대칭키 방식(관리자 비밀번호 = 암호화 키)\*\*으로 암호화하여 저장

외부인은 암호화된 상태의 데이터만 열람 가능

3\. 관리자 웹 대시보드

관리자가 로그인하면 비밀번호로 복호화되어 기록 열람 가능

블랙리스트에 추가 가능한 형식의 URL이라면 모두 모니터링 대상으로 등록 가능

접속 기록에는 UUID, 방문 URL, 시간, ID 가 함께 기록됨

관리자 페이지: http://d-plus.kro.kr:8080/login
